# Pre-requisites
1. Install NodeJS

---

### Mac
* http://nodejs.org/  
OR  
* https://github.com/creationix/nvm

### Windows
* http://nodejs.org

---

2. Install [Git](http://git-scm.com/downloads)
3. Install dependencies

```bash
npm install
```

# Run the project
```bash
npm start
```

# Run unit tests
```bash
npm test
```
